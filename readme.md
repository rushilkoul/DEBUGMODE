# DEBUGMODE  
*A first-person 3D puzzle game.*

> You are a developer on the verge of getting fired.  
> The tickets are overdue.  
> The build is broken.  



## About the Game

**DEBUGMODE** is a first-person 3D puzzle game where the player enters broken programs.  
Inside the program world, the player must solve puzzles and carry out tasks to fix all the bugs and push the code to Prod. (without review ofc)

Your goal is simple:
**Complete your pending tickets before the system (or your job) collapses.**



## üß† Core Gameplay

- Explore a glitchy developer workspace
- Solve puzzles by interacting with code execution flow
- Pet your dog (We ban any player that does not do this. it is law.)
